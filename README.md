# Portfolio 3D Interactif

Portfolio personnel développé avec React et Three.js, présentant une expérience utilisateur immersive et interactive.

## 🚀 Technologies Utilisées

- React 18
- Three.js / React Three Fiber
- Tailwind CSS
- Vite

## 🛠 Fonctionnalités

- Modèles 3D interactifs
- Animations fluides
- Design responsive
- Navigation intuitive
- Sections : About, Education, Experience, Projects, Contact

## 📦 Installation

```bash
# Cloner le projet
git clone https://github.com/hamza1br/hamza1br.github.io.git

# Installer les dépendances
npm install

# Lancer en développement
npm run dev

# Build pour production
npm run build
```

## 🌐 Déploiement

Le site est déployé sur GitHub Pages : [https://hamza1br.github.io](https://hamza1br.github.io)

## 📝 Structure du Projet

```
portfolio-website/
├── src/
│   ├── components/     # Composants React
│   ├── assets/        # Ressources
│   └── App.jsx        # Composant principal
├── public/            # Modèles 3D et assets statiques
└── dist/             # Build de production
```

## 🎨 Personnalisation

Le site utilise Tailwind CSS pour le styling et peut être personnalisé via :
- `tailwind.config.js` pour les thèmes
- `src/index.css` pour les styles globaux
- Composants individuels pour les styles spécifiques
